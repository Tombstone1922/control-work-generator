# Control Work Generator

MVP-прототип программного обеспечения для автоматизации формирования контрольных работ по рабочим программам дисциплин с применением нейросетевых и NLP-подходов.

Проект ориентирован на закрытое развертывание в контуре образовательной организации: рабочие программы дисциплин и контрольные материалы не требуется передавать во внешние публичные сервисы.

## Возможности MVP

- загрузка РПД в форматах DOCX, PDF, TXT;
- извлечение текста из документа;
- базовый анализ РПД: темы, компетенции, результаты обучения;
- генерация контрольных заданий по извлеченной структуре;
- оценка качества: покрытие тем, дубли, базовая релевантность;
- экспорт контрольной работы в DOCX;
- REST API на FastAPI;
- заготовка frontend-интерфейса на React + Vite.

## Запуск backend

```bash
cd backend
python -m venv .venv
# Windows
.venv\\Scripts\\activate
# Linux/macOS
source .venv/bin/activate

pip install -r requirements.txt
python run.py
```

API будет доступен по адресу:

```text
http://127.0.0.1:8000
```

Swagger UI:

```text
http://127.0.0.1:8000/docs
```

## Запуск frontend

```bash
cd frontend
npm install
npm run dev
```

## Основной сценарий работы

1. Загрузить РПД через endpoint `/api/programs/upload`.
2. Получить извлеченные темы, компетенции и результаты обучения.
3. Запустить генерацию через `/api/generation/run`.
4. Посмотреть отчет качества.
5. Экспортировать контрольную работу через `/api/export/docx/{session_id}`.

## Исследовательская идея

На этапе MVP используется гибридный генератор: структурный анализ РПД + правила + шаблоны. Архитектура выделяет нейросетевой модуль как отдельный компонент, чтобы далее подключить локально развернутую специализированную модель, обученную на реальных РПД и соответствующих контрольных работах.
